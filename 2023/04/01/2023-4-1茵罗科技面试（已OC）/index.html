2023-4-1茵罗科技面试（已OC）

2023-4-1茵罗科技面试（已OC）

JD

基础信息

RPA开发工程师
招聘人数：1-3(人)
薪酬福利：实习生3000+项目补贴，正式员工6000+项目补贴，报销跟进公司各岗位对应的内部报销政策

岗位职责

1.在RPA机器人自动流程化项目实施中，完成RPA的定制化程序开发
2.在开发过程中，确保产品的功能、性能和拓展满足交付条件
3.根据项目的需求和问题，持续优化RPA产品功能
4.定期对RPA进行效能重构，开发可用性高的新功能；完成上级交付的其余合理工作

任职要求

1.本科及以上学历，本科及硕士应届毕业生/实习生
2.一年以上开发经验且掌握至少一种编程语言，例如C#, JAVA，.在NET，Python，VB等技术，对RPA具有学习热情和职业生涯规划的学生优先
3.出色的问题分析和解决能力
4.熟悉软件开发流程和方法
5.熟悉RPA常用工具者任何一种，优先考虑（例如UiPath，BluePrism，Automation Anywhere或者其他 RPA产品）
6.做事积极主动，具备良好的沟通能力、协调能力，有较强的学习能力和抗压能力
7.能接受出差的工作性质，愿意接受学习新技术
8.具有较强的敬业精神、良好的合作精神和分析能力
工作地点：上海，杭州，苏州，花桥，成都，沈阳、北京、广州。

技术问题

  1. 介绍了RPA自动化机器人开发和公司情况（刚起步），问是否对此感兴趣？

python

  1. 使用过python吗？有哪些使用经验？

    答：简单使用过，数学建模中使用过，在SAP实习时用python处理表格、造测试数据

  2. 使用过哪些python库？做爬虫用哪些库了解吗？

    答：numpy基础库，pandas数据分析，matplotlib画图，scipy科学计算；爬虫库不太了解

    ANS:
    爬虫常用库有：
    lxml、beautifulsoup4 页面解析
    pyquery 语法类似于Jquery
    re 用于正则表达式
    requests http请求
    selenium 解决JS的渲染问题
    urllib3 Python HTTP库，安全连接池、支持文件post、可用性高。

Java

  1. 讲讲对枚举类型的了解
  2. 请介绍一下值类型、引用类型
  3. 进行异常处理的时候，接触过哪些exception类型？自定义异常了解吗？
  4. 介绍一下守护线程

数据库

  1. 请写一个SQL查询语句
  2. 在Mysql中，设置一个自增的属性id，开始表中的行：id=1、id=2、id=3，然后删除id=3的行，再加入一行，这时加入的id=3还是id=4？为什么会这样？

    ANS:id=4

  3. 介绍一下悲观锁 / 乐观锁的区别、具体实现
  4. B+树的优点？

网络

  1. GET和POST的具体传参方式以及结构
  2. cookie和session是什么？它们之间的关系和区别？

前端

  1. 接触过前端吗？

    答：用过三件套写简单的网页，使用过vue写简单网页，做测试的时候也与前端交互

  2. ajax了解吗？

体验

是HR、面试官一起的电话面试，没开视频。HR的态度比较差，很不耐烦，面试官态度比较好。
后面OC后提到SAP实习经历后，HR态度比较好，希望能够入职，但话里话外表示待遇一般。公司也刚成立，稳定性一般。
